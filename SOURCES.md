## Excluded / not used as result
- Per-domain composite-score specialist tables (W and S candidate rankings):
  rich but not load-bearing for the prefill/generation contrast; the paper uses
  only the domain-winner rows.
- E114-specific self-reference / phenomenological-register interpretation in the
  source repos' `CLAUDE.md` and `results.md` notes (a separate line of work):
  deliberately set aside. E114 appears only as the 35B philosophy generation
  winner, a worked example of dispersion, with no mechanistic claim.
- Second 35B run `20260415T214918Z`: shares the deterministic prefill capture with
  `20260408T235839Z` (identical prefill top-expert rows), so it is a re-analysis,
  not an independent replication; not cited as one.
- Raw `.npz` / `.npy` router tensors: not load-bearing for the winner-table claims.

## Conflicts / cautions noted
- The 122B source notes frame the generation analysis around finding an analog of
  one particular expert (E114). That framing is set aside; the paper reports only
  the pass-dependent concentration the same domain-winner tables support.
- `text-results.md` lists 122B generation global leaders (E0/E11/E5) which differ
  from the per-domain winners; the paper distinguishes "global top experts" from
  "domain winners" and uses the domain-winner tables for the dispersion counts.
- "Domain winner" is the least reliable layer (3 prompts/domain). Paper rests on
  the aggregate concentration contrast, not single-domain assignments.
